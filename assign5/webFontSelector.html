Web Font Selector
Choose a font… any font!  All of these fonts are web fonts embedded with @font-face and WOFF technology, so it shouldn't matter which font you pick.  Unforunately, I can only get the font changing to work in Chrome, but Firefox and Opera users should be able to see the font list and browse choices.
Your browser does not support <canvas>.  Sorry, that means you can't experiment with the fonts here.
